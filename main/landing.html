factorB

Сервис сверки Денежных Требований
между факторинговыми компаниями


Для чего?

  • factorB применяется для минимизации риска мошенничества в виде одновременного финансирования денежного требования в двух факторинговых компаниях
  • factorB сверяет денежные требования и сообщает о факте наличия денежного требования в нескольких факторинговых компаниях
  • Для сверки не требуется раскрытия никаких первичных данных. Нет риска утечки коммерческой/банковской тайны

Тестовая эксплуатация

Сервис запущен в тестовую эксплуатацию. Цель тестирования - подтверждение/опровержение актуальности проблемы двойного финансирования путём анализа портфеля уступленных денежных требований за 2015 год.

Бесплатно

Тест бесплатен для факторинговых компаний. Результат теста - сообщение об отсутствии или присутствии фактов наличия одного и того же денежного требования в портфеле другой факторинговой компании с указанием количества выявленных совпадений.
