factorB

сервис сверки денежных требований
между факторинговыми компаниями


Для чего?

  • factorB применяется для минимизации риска мошенничества в виде одновременной уступки и финансирования одного денежного требования в двух факторинговых компаниях
  • factorB сверяет денежные требования разных факторинговых компаний и сообщает о факте наличия одного требования в нескольких компаниях
  • для сверки не требуется раскрытия никаких первичных данных. Нет риска утечки коммерческой / банковской тайны

Тестовая эксплуатация

Сервис запущен в тестовую эксплуатацию.
Цель тестирования - подтверждение или опровержение актуальности проблемы двойной уступки.
Тестируется портфель уступленных денежных требований за 2015 год.

Бесплатно

Тест бесплатен для факторинговых компаний. Результат теста - сообщение о наличии / отсутствии совпадений в портфелях с указанием количества выявленных совпадений.
