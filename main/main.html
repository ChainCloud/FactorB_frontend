Инструкция

1 - Сформируйте исходный файл со столбцами вида ИНН1, ИНН2, Дата, Сумма....
Данные должны иметь кодировку UTF8.

2 - Посчитайте SHA256 хэш для каждой строки. Поля объединяются следующим образом: ИНН1+ИНН2+Дата+Сумма (без пробелов).
Пример EXCEL файла: здесь

3 - Скопируйте полученный столбец с SHA256 хэшами в отдельный файл формата CSV (каждый хэш с отдельной строки). Все остальные (исходные) данные не должны быть представлены в этом файле, чтобы избежать их раскрытие.

4 - Загрузите полученный файл в нашу систему.

5 - Через некоторое время вы получите отчет на указанный e-mail адрес.
